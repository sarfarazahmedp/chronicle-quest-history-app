# Chronicle Quest

A mobile-first gamified history learning app inspired by Paladin-style story learning.

Open `index.html` in a browser to run the prototype. Progress, XP, streaks, lesson scores, daily challenge state, and reminder preferences are saved in local storage.

## Included Loops

- Narrative micro-lessons with branching story choices
- Quizzes after every lesson
- XP, levels, streaks, boosts, and streak freezes
- Skill tree gates for eras and regions
- Daily challenge sprint with leaderboard
- Push-notification preference flow
